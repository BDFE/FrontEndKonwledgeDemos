三、 location.hash + iframe跨域 实现原理： a欲与b跨域相互通信，通过中间页c来实现。 三个页面，不同域之间利用iframe的location.hash传值，相同域之间直接js访问来通信。 具体实现： A域：a.html -> B域：b.html -> C域：c.html， 如下： a.html：(http://www.domain1.com/a.html) b.html：(http://www.domain2.com/b.html) c.html：(http://www.domain1.com/c.html) a与b不同域只能通过hash值单向通信， b与c也不同域也只能单向通信， 但c与a同域，所以c可通过parent.parent访问a页面所有对象。
